Déclaration d'Accessibilité de

La présente est la Déclaration d'Accessibilité officielle de . Si vous avez des questions ou des commentaires, n'hésitez pas à contacter l'administrateur du site : .

Clés d'Accès

La plupart des navigateurs supportent le saut vers un lien spécifique en tapant des touches définies sur le site. Sous le système d'exploitation Microsoft Windows, vous pouvez faire Alt + un touche définie; sous le système Macintosh de Apple, vous pouvez faire les touche Control + une touche définie.

Toutes les pages de ce site comportent leurs propres combinaisons de touches d'accès :

  • aucune combinaison de touches d'accès disponible

Conforme aux Standards

  1. Toutes les pages de ce site sont Bobby AAA-approved, et suivent toutes les directives du Manuel Bobby. C'est une question de jugement; bien des fonctions d'accessibilité peuvent être mesurées, mais beaucoup d'autres non. Toutes les directives ont été révisées et il est vraisemblable que toutes les pages de ce site en sont conformes.
  2. Toutes les pages de ce site suivent les directives WCAG AAA-approved, et sont conformes avec les directives de priorité 1, 2, et 3 du W3C. Encore une fois, c'est une question de jugement: plusieurs de ces directives sont intentionnellement vagues et ne peuvent être testées automatiquement. Toutes les directives ont été révisées et il est vraisemblable que toutes les pages de ce site en sont conformes.
  3. Toutes les pages de ce site sont conformes selon les termes de la Section 508, et suivent les directives de la Section 508 du gouvernement fédéral des États-Unis. Encore une fois, c'est une question de jugement. Toutes les directives ont été révisées et il est vraisemblable que toutes les pages de ce site en sont conformes.
  4. Toutes les pages de ce site sont conformes et passent les tests de validation XHTML 1.0 Strict. Ce n'est pas une approximation, un programme peut déterminer de façon certaine si une page est valide XHTML. Par exemple, vous pouvez cliquer sur ce lien pour vérifier la validité XHTML de cette page.
  5. Toutes les pages de ce site ont une structure sémantique de définition de langage. Les attributs H2 sont utilisés pour les titres principaux, les attributs H3 pour les sous-titres. Par exemple, sur cette page, les utilisateurs de JAWS peuvent sauter à la prochaine section dans le cadre de l'accessibilité en appuyant sur les touches ALT+INSERT+3.

Aide à la Navigation

  1. Toutes les pages sont pourvues de liens rel=précédent(e), suivant(e), haut, et accueil aidant à la navigation sur les navigateurs texte. Les utilisateurs de Netscape 6 et Mozilla peuvent aussi tirer avantage de cette fonctionnalité ou ouvrant le menu Afficher et en sélectionnant Montrer/Cacher, la barre de navigation, Afficher lorsque nécessaire (ou toujours afficher).
  2. Toutes les pages incluent une boîte de recherche (clé d'accès 4). Des options de recherche avancées sont disponibles sur la page de ">Recherche avancée.

Liens

  1. La plupart des liens ont un attributs title décrivant le lien plus en détail, sauf dans les cas où le texte du lien décrit déjà de façon complète la cible du lien (comme le titre d'un article).
  2. Les liens sont écrits de façon à donner un sens lorsqu'ils sont hors-contexte.

Images

  1. Toutes les images de contenu de ce site utilisent l'attribut de description ALT. Les images purement décoratives incluent un attribut ALT nul.
  2. Les images complexes incluent l'attribut de description longue LONGDESC, ou une description liée pour expliquer la signification de chaque image aux lecteurs non-voyants.

Design Visuel

  1. Ce site utilise des feuilles de style pour son rendu visuel.
  2. Ce site n'utilise que des tailles de police de caractère relative, compatibles avec les options de "taille de texte" des navigateurs.
  3. Si votre navigateur ou votre outil de navigation ne supporte pas les feuilles de style, le contenu de chaque page sera quand même lisible.
