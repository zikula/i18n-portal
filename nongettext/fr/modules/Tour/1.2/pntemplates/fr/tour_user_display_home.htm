Lancez-vous tout de suite !

Vous pouvez accéder au panneau d'administration en cliquant sur le lien Administration (situé aussi en haut de page). Si vous vous déconnectez, ce lien n'apparaîtra plus et vous ne pourrez y accéder à nouveau qu'en vous authentifiant via la page d'authentification.

Vous n'êtes actuellement pas identifié. Vous pouvez vous identifier en utilisant les identifiants d'administration que vous avez défini durant le processus d'installation en utilisant le bloc d'identification (à droite, normalement), ou alors utiliser la page d'authentification. Une fois que vous serez authentifié, vous pourrez voir un lien vers le panneau d'administration dans le menu principal. (situé en haut de page).

Lorsque vous serez prêt, vous pourrez enlever ce module et commencer à travailler sur votre site...

Pour ce faire, allez simplement dans les 'Préférences' situées dans l'onglet 'Système' du panneau d'administration. Pour afficher un module différent sur la page d'accueil, modifiez le module de démarrage. Vous pourrez aussi désactiver ce module via le module 'Modules' dans le panneau d'administration quand vous aurez fini, mais en attendant cette documentation restera disponible à partir de ce lien .
