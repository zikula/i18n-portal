Les Blocs

Pour ajouter un lien dans le menu de navigation du module que vous venez d'installer, il vous faut comprendre le concept de bloc. En continuant avec la métaphore du bureau de votre ordinateur, les blocs s'apparrentent à des gadgets vous permettant d'afficher la température, l'heure, le nombre de messages en attente, ... Les blocs offrent d'autres informations et complètent le contenu principal (Les modules). Le Menu principal et le formulaure d'authentification sur les côtés de votre site sont des exemples de blocs.

Vous pouvez avoir des blocs déstinés par exemple à l'affichage de 5 dernières nouvelles, les nouveaux téléchargements, les derniers messages du forum ou encore les utilisateurs en ligne. Chaque nouveau module que vous installez peut vous apporter un certain nombre de blocs que vous pouvez placer sur votre site.

Maintenant, pour ajouter un nouveau lien dans votre menu de navigation, vous devez aller dans la ">gestion des Blocs du panneau d'administration et le modifier. Cliquez sur le bouton modifier à droite du menu principal pour voir un formulaire permettant de le modifier. A partir de là, vous pourrez ajouter un nouveau lien vers votre nouveau module ou supprimer des liens que vous ne désirez plus voir apparaître. Chaque bloc dispose de son propre formulaire qui vous permet de le configurer en fonction de vos besoins et de vos préférences.

Position des blocs

Vous pouvez organiser les blocs à tout moment et de la manière que vous désirez : #blockpositions">Position des blocs. Votre installation a différentes positions par défaut ('left', 'right' et 'center') auxquelles vous pouvez assigner de nouveaux blocs ou des blocs déjà existants. Par exemple, si vous désiez un bloc en haut de votre site vous devez :

  1. Créer une position de bloc appelée (par exemple) 'topbloc' dans le panneau d'administration des blocs.
  2. Inclure la position dans le calque de votre thème.
  3. Créer un nouveau bloc avec le contenu que vous désirez afficher.
  4. Assigner au nouveau bloc à la position 'topbloc'.

Vous pouvez créer autant de blocs que vous le souhaitez et les inclure dans vos calques de thèmes comme bon vous semble.
